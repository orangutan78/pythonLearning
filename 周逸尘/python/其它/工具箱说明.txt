工具箱.exe是由工具箱.py打包的一个文件。
.py及.exe文件打开前请注意：
    将random object.roj.txt与文件置于同一文件夹内。（否则取随机值-取随机字符时读取/写入文件时会弹出错误窗口）
.py文件打开前请注意：
    打开cmd或Visual Studio Code等其他终端，输入如下内容（前提是有pip）：
        pip install pymouse
        python -m pip install --upgrade pip     #（可能要重试）
        pip install pyuserinput
        pip install pypiwin32
        #（以上是pymouse模块的安装）
        pip install keyboard
        #（以上是keyboard模块的安装）
        #（若不下载模块，运行鼠标连点器时会报错；如果模块已有则无需安装）
        等待每项安装完毕
本程序分为九个功能（3个大功能，3个中功能，3个小功能）：
介绍（按本人认为功能性、实用性进行排行）：
    大功能：
        1)取随机值：
            （1）取随机数：
                该功能可以抽取规定范围内的随机整数及1~5位小数。
            （2）取随机字符：
                该功能可以抽取规定的随机字符，还可以保存在文件里，方便下次抽取。
        2)进制转换：
            该功能可以转换2~36进制的数字。（算法系本人编写）
        3)鼠标连点器：
            该功能可以通过按下快捷键来实现规定时间频率鼠标在当前位置的连续点击。
    中功能：
        4)节拍器：
            该功能可以根据给定速度（BPM）及拍数发出节拍声音（可调节）。
        5)计时器：
            该功能可以按下按钮和快捷键来实现计时及其中的暂停、停止、计次等功能。
        6)计算器：
            该功能可以根据给定数字和运算方式来实现加减乘除、乘方、开根等运算（开根算法系本人编写）
    小功能：
        7)凯撒密码转换器：
            凯撒密码：凯撒密码是一种替换加密的技术，明文中的所有字母都在字母表上向后（或向前）按照一个固定数目进行偏移后被替换成密文。例如，当偏移量是3的时候，所有的字母A将被替换成D，B变成E，以此类推。
            该功能可以根据给定明文（英文）转换成凯撒密码。
        8)计算阶乘:
            该功能可以根据给定数字计算其阶乘。
        9)判断质数合数：
            该功能可以根据给定数字判断其为质数合数。
